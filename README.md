The ClearWater Conservancy map interface project is an online map interface that is built to show off all the conservation efforts as an aggregate to their current and future clients. The map interface is built on Mapbox and complete with easily accessible project descriptions and key facts suited to their needs. The map itself provides a clear view of all the sites categorized in the overall area that ClearWater conservancy operates in. The layers show off the local terrain, vegetations, and infrastructure to help users orientate themselves. There is also satellite view provided to help people see the whole area better.
